# Lumière Beauty — Estética & Bienestar

Landing page profesional para una esteticista autónoma, desarrollada como proyecto de portafolio y propuesta para cliente real.

🔗 **[Ver proyecto en vivo](https://melinarivera.github.io/lumiere-beauty/)**

---

## Funcionalidades

- Hero con imagen real, tarjetas flotantes animadas y estadísticas
- Sección de servicios con fotos, precios y duración
- Sección "Sobre mí" con valores y experiencia
- Testimonios de clientas
- Formulario de reserva de citas con selección de servicio, fecha y hora
- Confirmación visual al enviar la solicitud
- Botón de contacto directo por WhatsApp
- Navbar con scroll activo y logo diamante SVG
- Diseño 100% responsive

## Lo que aprendí con este proyecto

- Diseño orientado a un cliente real con necesidades concretas
- Uso de Google Fonts con tipografía editorial (Cormorant Garamond)
- Formulario de reserva interactivo con validación en JavaScript
- Animaciones CSS con `@keyframes` para badges flotantes
- Paleta de color femenina y cohesiva con variables CSS
